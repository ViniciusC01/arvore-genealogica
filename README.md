# Árvore-Genealógica

### A proposta principal desse projeto é criar uma classe Pessoa que contém os seguintes atributos:

~~~java
String nome;
~~~

~~~java
Pessoa antecessor;
~~~
</br>


### E a partir desses dados, criar uma relação hierárquica entre sucessor e antecessor utilizando o conceito de TreeSet.
